POST Teacher create

Overview

This API is to create teacher with allocated student details.

API details

Header Parameter

  • SecretKey string

    API Secret Key based on API settings

  • SchoolId string

    School Id For API based on API settings

Request

URL: /Teachers/Create

Body
{
  "id": 0,
  "firstName": "string",
  "lastName": "string",
  "age": 0,
  "email": "string",
  "class": 0,
  "students": [
    {
      "studentid": 0,
      "firstName": "string",
      "lastName": "string",
      "age": 0,
      "gender": "string"
    }
  ]
}
Sample Value
{
  "id": 204,
  "firstName": "Kumar",
  "lastName": "Babu",
  "age": 35,
  "email": "baule@xyz.com",
  "class": 8,
  "students": [
    {
      "studentid": 5,
      "firstName": "mari",
      "lastName": "alex",
      "age": 21,
      "gender": "male"
    }
  ]
}
Field Description

  • ID int

    Teacher ID

  • firstName string

    Teacher’s first name

  • lastName string

    Teacher’s last name

  • age string

    Age of the teacher

  • email string

    Email address of the teacher

  • class string

    Class of the teacher

  • students: Students information

    • studentid int

      Students unique ID

    • firstName string

      First name of the student

    • lastName string

      Last name of the student

    • age int

      Age of the student

    • gender string

      Gender of the student

Response
Body
{
  "IsSuccess": true,
  "Data": "string",
  "Errors": "string"
}
Response Success
{
    "IsSuccess": true,
    "Data": [
        {
            "id": 200,
            "firstName": "Kumar",
            "lastName": "Babu",
            "age": 35,
            "email": "baule@xyz.com",
            "class": 8,
            "students": [
                {
                    "studentid": 5,
                    "firstName": "mari",
                    "lastName": "alex",
                    "age": 21,
                    "gender": "male"
                }
            ]
        }
    ],
    "Errors": null
}
Response Failure

In case of errors, the API will respond with an appropriate HTTP status code and a JSON error message.

{
  "IsSuccess": false,
  "Data": null,
  "Errors": [
    {
      "Code": "INV0001",
      "Message": "Invalid Student ID"
     }
  ]
}
Field Description

  • IsSuccess string

    True/False

    If the value is True then the Data section will receive data from the external system. Error section will be NULL.

    If the value is False, then the Data section will be null. The Error section will be the list of validation based on which the teacher creation was failed.

  • Data

    • ID int

      Teacher ID

    • firstName string

      Teacher’s first name

    • lastName string

      Teacher’s last name

    • age string

      Age of the teacher

    • email string

      Email address of the teacher

    • class string

      Class of the teacher

    • students: Students information

      • studentid int

        Students unique ID

      • firstName string

        First name of the student

      • lastName string

        Last name of the student

      • age int

        Age of the student

      • gender string

        Gender of the student

  • Error

    • Code string

      Displays the error code

    • Message string

      Displays the error message

Error Codes and Messages

Use your error code or message to troubleshoot problems with the API.

Error Code: INV0001

Message: Invalid Student ID

Problem: Student ID does not exist

Error Code: INV0002

Message: Invalid Teacher ID

Problem: Teacher ID does not exist
